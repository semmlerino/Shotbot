ROOT CAUSE DEPENDENCY CHAIN
---------------------------

     pytest-xdist (parallel workers)
              |
              +--- Worker A          +--- Worker B
              |    Test File X       |    Test File Y
              |    ProgressManager   |    Launcher Manager
              |    _instance = {...} |    _instance = None (reads from A!)
              |                      |
              v                      v
        Module-level state persists in Python's import cache!
              |
              +--- Singleton._instance = {state}
              |    _operation_stack = [stale_data]
              |    _status_bar = <old_widget>
              |
              +--- Qt Resource: QPixmapCache accumulates, QThreadPool has pending work
              |
              +--- Module Cache: clear_all_caches() dict still has old entries
              |
              +--- QThreadPool: globalInstance() shared, threads still running
              |
              +--- Signal connections: deferred slots not processed
              |
              +--- ProcessPoolManager: _instance with active workers

Result: TESTS FAIL in parallel because of contamination
