## Safe Patterns - APPROVED

### Pattern 1: Fixture Usage (RECOMMENDED)
```python
@pytest.fixture(scope="session")
def qapp():
    """Session-scoped - shared across all tests."""
    app = QApplication.instance()
    if app is None:
        app = QApplication(["-platform", "offscreen"])
    return app
```
**Location**: tests/conftest.py (lines 53-97)

## DANGEROUS Patterns - PROHIBITED

### Pattern 1: Module-Level Creation
```python
# ❌ DANGEROUS - NEVER DO THIS
from PySide6.QtWidgets import QApplication

app = QApplication([])  # Executes at module import!

def test_something():
    pass
```
**Why forbidden**: Executes at import time, breaks parallel execution

---

### Pattern 2: Unguarded Top-Level
```python
# ❌ DANGEROUS
from PySide6.QtWidgets import QApplication

QApplication([])  # Side effect at module level!
```
**Why forbidden**: Module-level side effects prevent test isolation
